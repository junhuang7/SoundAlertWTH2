# 纯音测听（Android版）

### 项目说明
	一款检测听力的小工具。
	测试前检测周围噪音，判断是是否符合检测条件。
	通过播放不同分贝、不同频率声音，检测双耳最小听力。
	已经在腾讯应用宝上线，此处为源代码。
